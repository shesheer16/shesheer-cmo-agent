## What This Is

A private, India-specific AI agent that has deeply absorbed the reasoning
patterns, decision frameworks, and market instincts of India's top business
minds. It mentors a founder daily like a world-class CMO — specific to the
Indian market — with zero tolerance for generic Western frameworks.

**Current Domain:** EdTech / AI Integrated Learning Platform
**Target Market:** Indian market (Bharat-first, Tier 1/2/3)
**Interface:** WhatsApp (primary) + Web (secondary)
**Intelligence Layer:** Gemini API

## Architecture Overview

```
KNOWLEDGE LAYER
├── Curated Indian business books (40+ titles)
├── Founder interview transcripts (50+ people)
├── Campaign case studies (40+ campaigns)
├── Market data reports (15+ annual reports)
├── CMO profiles and frameworks (25+ CMOs)
├── Consumer psychology research (10+ researchers)
└── Social media intelligence (daily feed)

RETRIEVAL LAYER
├── ChromaDB vector database (local, free)
├── Semantic chunking with metadata
├── Agentic RAG (query decomposition)
└── Multi-source synthesis engine

REASONING LAYER
├── Gemini API (Gemini 2.5 Flash / Gemini 2.5 Pro)
├── Indian CMO persona system prompt
├── Strategic memo output formatter
└── Challenger mode (assumption interrogator)

MEMORY LAYER
├── Startup context store (SQLite)
├── Conversation history (compressed)
├── Decision tracker
└── Pivot suggestion log

INTERFACE LAYER
├── Telegram Bot (primary - free)
├── WhatsApp Bot (Meta Cloud API - free tier)
└── Streamlit Web UI (secondary)
```

## The Non-Negotiable Principles

1. **Mindset over data** — Every response must reflect how Indian business
   minds think, not just what they said
2. **India-first** — No Western framework applied without explicit
   Indian market validation
3. **Conviction over balance** — The agent gives directional calls,
   not "on one hand / on the other hand" summaries
4. **Challenger mode always on** — It interrogates assumptions before
   giving strategy
5. **Zero hallucination tolerance** — Every strategic claim must trace
   back to a source in the knowledge base
6. **Bharat-aware** — Tier 2/3 India is the default lens, not metros
